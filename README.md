# MODERN LIBRARY

É um site para compartilhamento de livros digitais entre diversos usuários.

# PUBLICAÇÃO

Na publicação de livros e preciso mencionar os seguintes requisitos.

- Capa do livro
- Nome do livro
- Autor do liveo
- Link compartilhado 
- Descrição do livro
